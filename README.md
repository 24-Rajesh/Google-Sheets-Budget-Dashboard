# Google-Sheets-Budget-Dashboard

## Project Overview
Created an interactive budget dashboard to track income, expenses, and savings.

## Key Features
- Monthly income & expense tracking
- Savings analysis
- Charts and visual reports
- Automated formulas
- KPI summary cards

## Tools Used
- Google Sheets
- Charts
- Formulas
- Conditional Formatting
- Data Analysis
